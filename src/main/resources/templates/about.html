프로젝트 개요

이 프로젝트는 Spring Boot와 Thymeleaf, 그리고 Payload CMS를 활용한 현대적인 웹 애플리케이션입니다.

주요 기능

  • Spring Boot 기반의 안정적인 백엔드
  • Thymeleaf를 이용한 서버사이드 렌더링
  • Payload CMS를 통한 동적 컨텐츠 관리
  • Bootstrap을 활용한 반응형 디자인

기술 스택

백엔드
  • Java 17
  • Spring Boot 3.2.0
  • Spring Web
  • Spring WebFlux (CMS 연동)
  • Spring Cache
프론트엔드
  • Thymeleaf 템플릿 엔진
  • Bootstrap 5
  • JavaScript
  • CSS3

CMS

  • Payload CMS (Headless CMS)
  • MongoDB 데이터베이스
  • JSON 기반 동적 컨텐츠
  • 실시간 API 연동
