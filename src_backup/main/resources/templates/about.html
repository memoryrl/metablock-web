프로젝트 개요

이 프로젝트는 Spring Boot와 Thymeleaf를 기반으로 한 웹 애플리케이션입니다.

주요 기술 스택

  • Spring Boot 3.2.0 - 메인 프레임워크
  • Thymeleaf - 템플릿 엔진
  • Spring Data JPA - 데이터 액세스
  • H2 Database - 개발용 인메모리 데이터베이스
  • Bootstrap 5 - UI 프레임워크
  • Lombok - 코드 간소화

기능

  • 반응형 웹 디자인
  • Thymeleaf 템플릿 시스템
  • 개발자 도구 및 핫 리로드
  • H2 콘솔 (개발용)
